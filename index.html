Library Kustom Documentation

Panduan penggunaan library JavaScript sederhana

Pendahuluan

Library ini dirancang untuk memberikan fungsionalitas seperti jQuery dalam bentuk yang lebih sederhana dan lebih ringan, memungkinkan pengguna untuk menambahkan event listener, mengatur properti CSS, serta mengelola kelas CSS pada elemen HTML dengan mudah.

Instalasi

Tidak diperlukan instalasi khusus. Cukup sertakan skrip library ini dalam tag <script src="path/your_file"></script> di HTML Anda.

Fungsi Konstruktor

function $(selector) {
if (!(this instanceof $)) {
return new $(selector);
}
this.elements = document.querySelectorAll(selector);
}

Memilih elemen berdasarkan selektor CSS yang diberikan.

Metode

$.prototype.on(event, callback)

$.prototype.on = function (event, callback) {
this.elements.forEach(function (element) {
element.addEventListener(event, callback);
});
return this;
};

Menambahkan event listener ke elemen yang dipilih.

$.prototype.css(property, value)

$.prototype.css = function (property, value) {
this.elements.forEach(function (element) {
element.style[property] = value;
});
return this;
};

Mengatur properti CSS pada elemen yang dipilih.

$.prototype.classAdd(className)

$.prototype.classAdd = function (className) {
this.elements.forEach(function (element) {
element.classList.add(className);
});
return this;
};

Menambahkan kelas CSS ke elemen yang dipilih.

$.prototype.classRemove(className)

$.prototype.classRemove = function (className) {
this.elements.forEach(function (element) {
element.classList.remove(className);
});
return this;
};

Menghapus kelas CSS dari elemen yang dipilih.

$.prototype.classToggle(className) 

$.prototype.classToggle = function (className) {
this.elements.forEach(function (element) {
element.classList.toggle(className);
});
return this;
};

Toggle (aktif/nonaktif) kelas CSS pada elemen yang dipilih.

Contoh Penggunaan


document.addEventListener("DOMContentLoaded", function() {

$('selector').classAdd('active');
$('selector').css('color','red');
$('selector').on('click', function() {
$('selector').classToggle('clicked');
});
});

Menggunakan library kustom untuk menambahkan event listener, mengatur properti CSS, dan mengelola kelas CSS pada elemen HTML.
